# FMP Client
React native app for scrapping fmp.es website.


## Requirements
* nvm
* [expo](https://github.com/expo/expo/tree/main/packages/%40expo/cli)

## Installation
* Run `nvm use`
* Run `npm i`
* Install expo app on your device https://expo.dev/client
## Run project
* run `expo start` o `npx expo start`
